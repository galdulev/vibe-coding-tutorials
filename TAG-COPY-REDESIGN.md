1. **Specific > Generic**

   - âœ… "one coffee break"

2. **Show Don't Tell**
   - âŒ "easy to use"
   - âœ… "dashboard by lunch"

   - âŒ "great AI"

## Implementation Notes

### Typography for Tags
```css
.tag {
    font-size: 18px;           /* Bigger = confidence */
    font-weight: 500;          /* Not too heavy */
    letter-spacing: -0.01em;   /* Tighter = modern */
    text-transform: lowercase; /* Casual, approachable */
}
```

### Emoji Rules
- Use sparingly (1 per tag max)
- Match the vibe (âš¡ = fast, ğŸ’« = magical)
- Consider contrast on your bg color
- Test on mobile (some emojis render weird)

### Avoid
- ALL CAPS (too shouty)
- Too many words (3-4 words max)
- Jargon without context
- Trying too hard (be cool, not "hello fellow kids")
